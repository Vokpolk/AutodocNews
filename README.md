# Autodoc News App

Тестовое задание для iOS-разработчика. Приложение для просмотра новостей компании Autodoc.

## Стек технологий
* **Архитектура:** MVVM (Model-View-ViewModel)
* **UI:** Полностью кодом (No Storyboards), UICollectionView + Compositional Layout
* **Данные:** Diffable Data Source
* **Реактивность:** Combine
* **Сеть:** URLSession + async/await (без сторонних библиотек)
* **Изображения:** Кастомный ImageService с NSCache (без Kingfisher)
* **Адаптивность:** Поддержка iPhone и iPad

## Особенности реализации
* Реализована пагинация (бесконечный скролл).
* Обработка жизненного цикла ячеек (отмена загрузки картинок при переиспользовании).
* Отсутствие внешних зависимостей (чистый SDK).
* Индикация состояний загрузки (Activity Indicator) для контента.
